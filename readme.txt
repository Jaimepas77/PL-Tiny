El main principal está en la carpeta "implementacion", archivo Main.java.
Este main ejecuta la práctica tal y como se indica en el enunciado de la segunda parte.

Para poder ejecutar el analizador ascendente generado por java cup, se necesita incluir la librería runtime.

Los códigos de prueba, situados en la carpeta "pruebas", tienen un nombre de fichero siempre terminado por ".pas".
Aquellos que empiezan por "codigo" deben funcionar correctamente.
Aquellos que comienzan por "error" deben devolver un error o bien de ejecución o bien de algún procesamiento.
El archivo "ejemplo.pas" contiene el código de ejemplo dado en el enunciado.

Existe un segundo main contenido en el archivo "ejemploPrincipal.java", carpeta "pruebas".
Este segundo main pone a prueba la primera parte de la práctica.
